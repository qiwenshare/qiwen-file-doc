  details: 集成 vue-simple-uploader，实现文件分片上传，提高上传速度；准确识别已上传文件，达到秒传效果，以分片为维度，可实现断点续传
- title: 在线编辑文档

  details: 回收站可以轻松帮助找回误删文件，对回收站进行清理后文件才将彻底删除。
- title: 多种展示模式
  details: 提供了列表、网格、时间线模式，网格模式下支持手动调整图标大小
- title: 全局搜索
  details: 支持搜索文件名，快速定位文件并可以打开文件所处位置
- title: 文件分享
  details: 支持分享文件，设置过期时间，自动生成提取码；可查看个人分享过的文件列表
- title: 文件基本操作
  details: 支持拖拽上传文件，支持文件移动、重命名、解压缩、下载、删除、还原
- title: 文件预览
  details: 支持图片、视频、音频在线预览，支持 Office 文件在线预览
